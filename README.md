# 📦 Telegram Shop Bot

A complete e‑commerce bot for Telegram – clean, ready to sell anything.

## ✨ Features

- 🛍️ Product catalog with categories, images, price & stock  
- 🛒 Shopping cart  
- 💳 Manual payment (bank transfer) + QR code support  
- 📸 Receipt upload & admin verification  
- 📍 Delivery location sharing  
- 🚚 Order tracking with status updates  
- ⭐ Product reviews & ratings  
- 🔔 Back‑in‑stock alerts  
- 🌍 Multi‑language (English, Khmer, Chinese)  
- 🔐 Admin panel (add/edit/delete products & categories, broadcast, verify receipts, statistics)  
- 🧾 PDF / text receipt generation  
- 💾 Database backup & restore  

**No sample products are preloaded** – you add everything through the admin panel.
